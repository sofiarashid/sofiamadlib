Mad Lib Story

Build a story using prompt(), variables, template literals, and innerHTML. (No conditionals yet!)

Your Story

When your page loads and your JavaScript runs, your completed Mad Lib will appear here.

What to Do

  1. Use prompt() to ask the user for words (noun, verb, adjective, place, etc.).
  2. Store each answer in a clearly named variable using let or const.
  3. Create a story string using template literals (backticks) and insert variables with ${}.
  4. Use document.getElementById() to select #madlib-output and set .innerHTML to your story.
  5. Customize the CSS and overall style of this site to make it your own // fit the story you wrote.
  6. Delete these instructions when you have finished and replace them with an explanation of what this project is.

Bootstrap Note
